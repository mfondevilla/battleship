#  Battleship 🚢 Hundir la flota

Battleship es un pequeño juego de consola escrito en Python. El objetivo es adivinar la posición de los barcos generada aleatoriamente por el programa.

## Requisitos

- Python 3.0 o superior
- No requiere librerías externas

## Características

- Posiciones de los barcos generadas dinámicamente de forma aleatoria
- Variables configurables en un fichero variables.py
- Ejecución desde consola
- Modalidad de jugador vs. ordenador

## Estructura del proyecto
El código fuente del proyecto se encuentra organizado dentro del directorio `src`, mientras que los ficheros `main.py` y `README.md` permanecen en el nivel raíz del repositorio. Dentro de src el código se organiza en los siguientes directorios:  

`clases`: Definición de la clase tablero

`functions`: Definición de las funciones relativas a las acciones de: crear tablero, colocar barcos, disparar, c

`notebooks`:

`variables` : Definición de variables y constantes que se usarán en la diferentes funciones y clases del proyecto

            
## Guía de usuario
- Se requiere tener instalada una versión 3.0 de Python o superior
- Clonar el proyecto el local
- Abrir una terminal en la carpeta raíz del proyecto
- Ejecutar el programa con el comando: `python -m src.main`
- Seguir las indicaciones que irá marcando el juego por consola
